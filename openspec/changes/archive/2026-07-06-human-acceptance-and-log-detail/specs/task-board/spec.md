# task-board

## MODIFIED Requirements

### Requirement: 看板分列展示

系统 SHALL 按状态分列展示项目的任务，「验证中」为一等看板列，位于「进行中」与「已完成」之间。

#### Scenario: 查看看板
- **WHEN** 用户进入项目工作区
- **THEN** 系统按 待办/进行中/验证中/已完成 分列展示任务；「验证中」独立成列（区别样式），归档任务单独可见

#### Scenario: 待人工验收
- **WHEN** 任务执行完成进入验证中
- **THEN** 该任务停留在「验证中」列，直到管理员手动拖入「已完成」完成验收
